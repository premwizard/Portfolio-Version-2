# Iron Crown Portfolio - Rose Gold Edition

A premium, production-ready personal portfolio website built with React.js and Vite, showcasing a luxurious "Iron Crown – Rose Gold Edition" theme with advanced animations and responsive design.

![Portfolio Preview](https://via.placeholder.com/800x400/080507/f0ece8?text=Iron+Crown+Portfolio)

## 📖 Description

This portfolio website is crafted for modern developers and designers seeking to present their work with elegance and sophistication. Built using React 19, Vite, and Tailwind CSS, it features a royal aesthetic inspired by iron crowns and rose gold accents. The site includes smooth animations, interactive elements, and a fully responsive layout optimized for all devices.

## ✨ Features

- **Hero Section**: Dynamic landing with typewriter animation and call-to-action buttons
- **About Section**: Personal introduction with profile information and background
- **Skills Carousel**: Interactive carousel displaying technical skills with icons only
- **Projects Grid**: Showcase of portfolio projects in a responsive grid layout
- **Certificates Infinite Loop Carousel**: Seamless, continuous-scrolling carousel with modal previews and 3D tilt effects
- **Contact Form**: Functional contact form with validation and social media links
- **Responsive Design**: Fully responsive across desktop, tablet, and mobile devices
- **Smooth Scroll Animations**: Fade-in effects and micro-interactions throughout
- **Particle Background**: Animated dot background with theme colors
- **Glassmorphism Effects**: Modern glass-like UI elements with backdrop blur
- **Accessibility**: Focus states, ARIA labels, and keyboard navigation support

## 🎭 Animations Used

- **Typewriter Animation**: Dynamic text animation for the hero section name
- **Fade-in on Scroll**: Sections animate into view as user scrolls
- **Hover Effects**: Scale, glow, and color transitions on interactive elements
- **Carousel Auto-scroll Animation**: Smooth, continuous infinite loop scrolling
- **Shine Sweep Effect**: Animated shine effect on certificate cards
- **Modal Zoom Animation**: Smooth zoom-in/out for certificate modal previews
- **Floating Card Animation**: Subtle floating motion for project and certificate cards
- **Particle Animation**: Continuous movement of background particles

## 🚀 Tech Stack

- **React.js** (Vite): Modern frontend framework for building user interfaces
- **Tailwind CSS**: Utility-first CSS framework with custom theme configuration
- **Framer Motion**: Animation library for smooth transitions and micro-interactions
- **Embla Carousel**: Lightweight carousel library for infinite loop functionality
- **Lucide React**: Icon library for consistent, scalable icons
- **React Icons**: Additional icon set for enhanced UI elements

## 📝 Customization

### Personal Information
Update the following files with your information:
- `src/components/Hero.jsx` - Name and subtitle
- `src/components/About.jsx` - Bio and details
- `src/components/Contact.jsx` - Contact information and social links
- `src/components/Projects.jsx` - Your projects
- `src/components/Skills.jsx` - Your skills and proficiency levels

### Styling
- Colors are defined in `tailwind.config.js`
- Global styles in `src/index.css`
- Component-specific styles inline with Tailwind classes
